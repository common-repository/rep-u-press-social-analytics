== Installation ==

1. For manual code install upload and extract the Plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Click on the Social Analytics tab under the RepUPress SA tab.
4. Select a Date Range ing the upper right with the Date Range Picker.
5. Click Apply - That is it.  You can now see your analytic share data and efficiency score.

If you deactivate the plugin all settings will be retained until you uninstall.
